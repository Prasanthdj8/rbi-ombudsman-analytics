# Source PDFs

These are the three RBI Annual Reports of the Ombudsman Scheme used as the
sole data source for this project, included here for direct verification.

- `Annual Report of Ombudsman Scheme, 2022-23.pdf`
- `Annual Report of Ombudsman Scheme, 2023-24.pdf`
- `Annual Report of Ombudsman Scheme, 2024-25.pdf`

All three were downloaded from RBI's official publications page:
https://rbi.org.in/Scripts/AnnualPublications.aspx?head=Annual+Report+on+Banking+Ombudsman+Scheme

They are included unmodified so every figure in this project can be checked
directly against the original document rather than taking the numbers in
`SOURCES.md` on trust. See `../SOURCES.md` for the specific page references
and figures drawn from each report.
